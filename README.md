# Tetiana Hryn | Frontend Developer Portfolio

A modern responsive portfolio website created to showcase my frontend development skills, projects, and experience.

## 🌐 About The Project

This is my personal portfolio website where I present my skills, completed projects, and information about myself as a Junior Frontend Developer.

The main goal of this project was to create a clean, modern and responsive website that works well on all devices.

## 🚀 Features

- Responsive design for desktop, tablet and mobile devices
- Modern and clean user interface
- Mobile navigation menu
- Smooth scrolling between sections
- Active navigation links
- Scroll animations
- Projects showcase section
- Contact form with Formspree integration
- Social media links

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Formspree

To run this project locally:

1. Clone the repository:

```bash

```

Open the project folder.
Open index.html in your browser.

No additional installation is required.
